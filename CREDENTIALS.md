# AetherScan Credentials

This file stores the seeded local accounts for development and demo access.

## Local user accounts

- `admin@aetherscan.local` / `Admin123!`
- `engineer@aetherscan.local` / `Engineer123!`
- `tech@aetherscan.local` / `Tech123!`

## Notes

- These accounts are intended for local seeded data only.
- Do not expose these credentials in a production login screen.
- If you delete `.data/aetherscan.json`, the application will recreate the seeded data and these accounts will be available again.
